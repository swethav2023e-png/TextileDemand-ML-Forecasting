# TextileDemand

ML-Powered Textile Demand Forecasting Dashboard built using Python, Streamlit, XGBoost, and Scikit-Learn.

## Features
- Interactive analytics dashboard
- Demand forecasting using XGBoost
- Product-wise filtering
- KPI monitoring
- Real-time predictions

## Technologies Used
- Python
- Streamlit
- XGBoost
- Pandas
- NumPy
- Scikit-Learn
- Matplotlib

## Project Structure
- text.py : Main application
- model.pkl : Trained ML model
- Textile_sales_data.csv : Dataset
- requirements.txt : Dependencies

## How to Run

```bash
pip install -r requirements.txt
streamlit run text.py
```
